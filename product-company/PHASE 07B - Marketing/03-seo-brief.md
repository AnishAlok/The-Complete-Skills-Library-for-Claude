# seo-brief.md

> **Skill:** Produces an SEO content brief that gives a writer everything needed to produce a page that ranks: target keywords, search intent, suggested structure, competitor analysis, and internal linking.

---

## TLDR

An SEO content brief is the research and strategy document that precedes writing. It defines the target keyword, the search intent, the recommended content structure, the competitive landscape for that keyword, the word count range, and which internal pages to link to and from. A writer working from a good brief produces better content faster than a writer starting from a blank page with only a keyword.

---

## Topic Explanation

### Why Briefs Matter for SEO

A page that ranks for a competitive keyword is not an accident. It was planned: the keyword was researched, the intent was understood, the competitor content was analyzed, and the structure was designed to cover the topic more thoroughly and accessibly than what currently ranks.

Content produced without a brief tends to rank for nothing, because it was written for a vague audience with a vague intent at a vague depth.

### Search Intent Types

Understanding search intent is the most important input to an SEO brief. Google's algorithm is designed to serve the intent behind the query, not just the words in it.

**Informational:** the user wants to learn something. "How to write a job description." Content type: how-to article, guide, explainer.

**Navigational:** the user wants to find a specific page. "Notion templates." Content type: landing page for that specific resource.

**Commercial investigation:** the user is comparing options before a purchase. "Best project management tools." Content type: comparison page, roundup, versus article.

**Transactional:** the user is ready to act. "Sign up for Asana free trial." Content type: landing page, product page.

The brief must identify the intent and recommend content that serves it. Writing an informational blog post for a transactional keyword will not rank, no matter how well-written it is.

### SERP Analysis

Before writing the brief, analyze the top 5 to 10 results for the target keyword. Identify: what content types are ranking (article / list / video / tool), what topics they all cover (table stakes), what topics only the best results cover (differentiators), and what questions the top-ranking content does not fully answer (the gap the new content can own).

## Output Format

```
SEO CONTENT BRIEF

Page title (target): [working title including primary keyword]
URL slug: [/suggested-url-path/]
Date: [brief date] | Owner: [SEO/content lead]

TARGET KEYWORDS
Primary keyword: [exact keyword, monthly search volume, keyword difficulty]
Secondary keywords: [related terms to include naturally]
LSI terms: [semantically related terms that signal topical depth]
Long-tail variations: [specific question-based queries to address]

SEARCH INTENT
Intent type: [Informational / Commercial / Navigational / Transactional]
What the reader wants: [the specific question or need behind this search]
What a satisfying result looks like: [content type and depth that serves this intent]

SERP ANALYSIS
Content types ranking: [articles / lists / videos / tools / product pages]
Word count range of top 5: [min to max]
Topics all top results cover: [table stakes, must include]
Topics only top results cover: [differentiators that separate position 1 from 5]
Questions no current result fully answers: [the gap this content can own]
Featured snippet opportunity: [yes / no, and format: paragraph / list / table]

RECOMMENDED STRUCTURE
H1: [suggested headline including primary keyword]
Intro approach: [lead with stat / question / problem acknowledgment]

H2: [Section 1 - maps to table-stakes topic]
H2: [Section 2]
H2: [Section 3 - maps to differentiator gap]
H2: [Section 4]
H2: Conclusion / Summary

[Note any sections that should include a table, list, or visual for featured snippet capture]

CONTENT SPECIFICATIONS
Target word count: [X words based on SERP analysis]
Reading level: [accessible / technical / expert]
Tone: [reference brand-voice-guide.md]
Multimedia: [screenshot / diagram / video embed recommended]

SEO REQUIREMENTS
Title tag: [50 to 60 characters, includes primary keyword]
Meta description: [150 to 160 characters, includes keyword, drives click]
Primary keyword in: H1 / first 100 words / at least one H2 / naturally 3 to 5x in body
Image alt text: [describe what image alt text should cover]

INTERNAL LINKING
Link to this page from: [existing pages that should link here, with suggested anchor text]
Link from this page to: [pages this content should link to, with suggested anchor text]

EXTERNAL SOURCES TO CITE
[Specific studies, reports, or data sources that strengthen credibility for this topic]

WRITER NOTES
[Anything the writer needs to know that is not captured above:
specific angle to take, claims to avoid, SME to interview, competitor to not mention]
```

## Starter Prompt

```
You are an SEO strategist writing a content brief.

Target keyword: [primary keyword]
Monthly search volume: [if known]
Keyword difficulty: [if known from SEO tool]
Search intent: [what users want when they search this]
Existing top results: [describe or list what currently ranks]
Target audience: [role and stage of awareness]
Content goal: [TOFU traffic / MOFU lead / BOFU conversion]
Existing relevant content on site: [pages that could link to this]

Produce a complete SEO content brief.
SERP analysis section must identify at least one content gap the new piece can own.
Recommended structure must map to both table-stakes topics and differentiator gaps.
Internal linking must include specific anchor text suggestions.
```
